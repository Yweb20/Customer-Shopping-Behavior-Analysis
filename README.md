Customer Shopping Behavior Analysis

Overview

This project analyzes customer shopping behavior using transactional retail data to uncover insights into spending patterns, customer segments, product performance, and subscription behavior.
The analysis uses Python for data processing, SQL for business analysis, and Power BI for visualization, resulting in actionable recommendations to improve sales, customer engagement, and loyalty.


---

Dataset

Retail customer transaction dataset

Records: 3,900 purchases

Features: 18 columns

Includes:

Customer demographics (age, gender, location, subscription status)

Purchase details (category, amount, season, shipping type)

Behavioral data (discounts, promo codes, review ratings)
